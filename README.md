###### Value Share

#### 서비스설명

Value Share는 합리적인 소비를 추구하는 MZ세대를 위한 온라인 명품 대여 서비스입니다.

구매하지 않고 다양한 스타일을 시도해보며 가치소비를 실현할 수 있습니다.

고객의 일상에 명품이라는 선택지를 제안합니다.
